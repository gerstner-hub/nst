Detailed Description

template<class LV, class IT>
class nst::Screen::IteratorT< LV, IT >

Minimal iterator type for iterating over the current Screen content.

This is a simple iterator type that transparently handles the possible wrap-around in the ring buffer used in the Screen type.

Offering random access iterator semantics is hard, because we often cannot determine the correct context and cannot offer operator< & friends due to the ring buffer's nature. Thus this only implements output iterator semantics.
