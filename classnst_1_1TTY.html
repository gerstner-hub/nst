Detailed Description

(Pseudo) Terminal I/O.

This class covers the PTY/TTY interfacing towards the operating system. It's job is mainly the raw I/O handling and handling of low level TTY aspects.

It holds the sub process that is running on the terminal. It sends keyboard input and out-of-band data to the child process and receives data from it to display on the terminal.
