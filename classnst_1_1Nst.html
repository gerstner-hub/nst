Detailed Description

Nst main application class.

This type holds instances of all the other types that make up nst. It implements the main loop and is invoked from the main function of the program.

◆ pipeBufferTo()

void nst::Nst::pipeBufferTo ( const cosmos::StringViewVector  cmdline)

Pipe the current terminal contents (including scrollback) to the given program.

The data will be provided to the program's stdin. The call will run synchronously i.e. the terminal won't continue running until the child process has read all data and exits.

This is intended for graphical programs like gvim which daemonize and continue running the background.

Parameters
[in]cmdlineThe path to the program to run as well any additional command line parameters.
